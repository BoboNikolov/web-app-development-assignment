Weather Forecast

This page sends a request to the OpenWeatherMap web service and retrieves a JSON forecast for the city you enter. The response is parsed in JavaScript and displayed below in a readable format.

Check 5-Day Forecast

Enter a city name and view the upcoming forecast using data from the JSON response.

How This Page Works

When you enter a city and click Get Forecast, the application builds a URL for the OpenWeatherMap API using the city name and an API key. JavaScript then uses the fetch() function to send an HTTP GET request.

The response is returned as JSON. This JSON is parsed and key fields such as date/time, temperature, ‚Äúfeels like‚Äù value, humidity and a text description of the weather are displayed in a table. Basic form validation and error handling are also implemented so that invalid inputs and failed requests are clearly communicated to the user.
